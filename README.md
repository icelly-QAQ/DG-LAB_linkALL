# 郊狼万能控制器
当前版本 v0.1.0

## 🚀 快速开始

1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**：
   ```bash
   python src/main.py
   ```

3. **连接设备**：按照界面提示连接你的DG-LAB设备

4. **开始体验**：选择你喜欢的插件，开始你的电击之旅！
